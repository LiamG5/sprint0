Player Controls:
Move: WASD, Arrows (changes facing)
Attack: Z or N
Use Item: 1, 2, 3
Take Damge: E

Inventory Controls:
Open / Close Inventory: B
Move selection in inventory: Arrow Keys
Equip highlighted item to B slot: Enter
Use equipped B-slot item in gameplay: 2

Block/Obstacle Controls
Cycle current block backward: T
Cycle current block forward: Y

Item Controls:
Cycle current item backward: U
Cycle current item forward: I

Enemy/NPC Control:
Cycle current enempy/NPC backward: O
Cycle current enempy/NPC forward: P

Other Controls:
Quit: Q
Reset to initial state: R
Muisc: m
Inventory: b
